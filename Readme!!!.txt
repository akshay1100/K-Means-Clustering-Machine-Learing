This program is written in java.

Instrution to execute:
1.Open Cmd , and go the path where the "kmeans.java" and Data1 file is saved.
2.type: "javac kmeans.java" .
3."java kmeans <no of clusters> <input file> <output file>".
ex: java kmeans 3 Data1.txt output1.txt

**Input file: Data1.txt

4.The output text file will be generated in the same folder with the name given in arguemnt.
